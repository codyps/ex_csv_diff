# CsvDiff

A elixir wrapper around the [csv-diff rust crate](https://crates.io/crates/csv-diff)

## Installation

The package can be installed by adding `csv_diff` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:csv_diff, "~> 0.3.0"}
  ]
end
```

Docs can be found at <https://hexdocs.pm/csv_diff>.
